Get CADastra

This software has been designed as a freeware to aid surveyors in compiling JRJ files with ease.
Here are some features that have been included;

  • Custom file extension, easy to use and share
  • Nice and easy user interface
  • Plots Index and working Diagrams
  • Export index diagram in google earth
  • Export entire JRJ in one PDF file
  • Auto-compile: Computes entire JRJ with no adjustments needed
  • Dymanic: Supports over 1,000 stations and over 500 parcels
  • Free and easy Coordinate Conversion Software
  • Quick data Extraction: Quickly extract data from AutoCAD
  • Saves your Computational time by 60%
  • Inset Management: Manage parcels enclosed in other parcels
  • Added all districts in index Computations
  • Added alternative units(feet) to index diagram
  • And many many more. Tryout
For detailed video tutorials and refferences go to Youtube for more help
